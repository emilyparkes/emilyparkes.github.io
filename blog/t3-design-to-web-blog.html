What a responsive site is, and why responsiveness is important.

A responsive site is one that adapts the layout and/or content to fit different browser sizes. It is important because a responsive site is one that is more easily viewed across devices, creating a better user experience. None of the content will be unavailable or cut off.

What mobile first design is, and why it's important.

A mobile first design is when a site was created with the mobile phone size layout in mind first, using what's called a Media Query. Media Queries change the layout of a site using the 'min-width' label to tell the browser that anything larger than this 'min-width' uses the following layout descriptions.This means that mobile and other smaller devices can load quicker and don't need to proccess all the css before selecting what applies to the screen size.

What frameworks are, and their pros and cons.

Frameworks are basic html or css layouts that can save you from typing out lots of code yourself. This can be very helpful but also can be a problem if you are wanting to be more creative. It mainly depends on what frameworks you use and how.

What a wireframe is and why we use it.

A wireframe is a blueprint for a webpage, it shows us the layout that we want to achieve and how this will translate into html and css divisions. This makes it easier to see how we should be coding our html and css to create our desired look.
