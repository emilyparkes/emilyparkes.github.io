JavaScript Language

29 Nov 2017

The following are uses of syntax:

Parenthesis () - e.g Execute a function var result = doSomething()

Parenthesis' are used to give information to the function. They also are used in mathmatical equations like how they're used in algebra. They are also used for console logging and containing conditions for different typyes of loops.

Brackets []

These are used for multiple items listed in an array or for strings, as well as locating a position/item in an array.

Braces {}

Braces - or curly brackets, are used to enclose blocks of code in functions and loops.

Single Quotes ' ' and Double Quotes " "

There isn't much difference between single and double quotes, both can be used to define a string - the only difference being you can use a double quote inside a single quote.
